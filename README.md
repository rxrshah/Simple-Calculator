# Simple Calculator (HTML, CSS, JS)


A minimal calculator web app you can host on GitHub Pages. Supports basic arithmetic, percent, decimal, keyboard input, backspace and clear.


## Files


- `index.html` — the page
- `styles.css` — styling
- `script.js` — calculator logic


## Deploy on GitHub Pages

1. Create a new GitHub repository and push these files.
2. In repository settings -> Pages, choose `main` branch and `/ (root)`.
3. Visit the provided URL.
