# Simple Calculator (HTML, CSS, JS)


A minimal calculator web app you can host on GitHub Pages. Supports basic arithmetic, percent, decimal, keyboard input, backspace and clear.


## Files


- `index.html` — the page
- `styles.css` — styling
- `script.js` — calculator logic
